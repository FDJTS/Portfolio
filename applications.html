تطبيق المسلم

تطبيق إسلامي شامل للهاتف الذكي

الوصف:

تطبيق المسلم هو تطبيق إسلامي متقدم مصمم خصيصاً للمسلمين الباحثين عن تطبيق شامل يجمع بين الأدوات الدينية والعملية. يوفر التطبيق تجربة مستخدم سلسة وسهلة الاستخدام مع واجهة احترافية وتصميم حديث.

الميزات الرئيسية:

  • أوقات الصلاة الدقيقة حسب موقعك الجغرافي
  • جدول صلوات شامل ومفصل
  • متصفح القرآن الكريم الكامل
  • الأذكار اليومية والمأثورة
  • التقويم الهجري مع تحويل التواريخ
  • بوصلة القبلة بتقنية البوصلة الذكية
  • إشعارات ذكية قابلة للتخصيص
  • تحديد موقع دقيق بتقنيات GPS متقدمة
  • دعم الوضع الليلي والفاتح
  • إعدادات متقدمة قابلة للتخصيص

المنصات المدعومة:

متاح حالياً على نظام Android. يتم العمل على إصدارات iOS و Web.

مجموعة التقنيات:

React Native Expo TypeScript EAS Build Geolocation API SQLite

هيكل التطبيق:

  • الشاشة الرئيسية - عرض أوقات الصلاة الحالية
  • جدول الصلوات - عرض مفصل لأوقات جميع الصلوات
  • القرآن الكريم - متصفح تفاعلي للنصوص القرآنية
  • الأذكار - مجموعة شاملة من الأذكار المأثورة
  • التقويم الهجري - تتبع التواريخ الإسلامية
  • بوصلة القبلة - تحديد اتجاه القبلة بدقة
  • الإعدادات - تخصيص كامل للتطبيق
