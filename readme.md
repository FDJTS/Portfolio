# FDJTS | Portfolio

My personal website to showcase my programming projects and practical tutorials in programming and game development.

## What will you find here?

- **Home:** A brief overview of who I am and links to the most important tutorials.
- **Projects:** An auto-refreshing display of my latest GitHub projects with descriptions and links.
- **Tutorials:** Practical guides with images, code snippets, and videos (e.g., Godot and Arch Linux guides).
- **Contact:** A simple contact form that allows you to send me a message directly.

## Technologies Used

- HTML5, CSS3 (Light and Dark mode support)
- JavaScript (Interactive effects and automatic GitHub project loading)
- Responsive design with full bilingual support.

## How to Use

1. You can browse the site directly from any modern browser.
2. All images and files are located within the repository.
3. To modify or add tutorials or projects, edit the HTML files or add new images to the same folder.
